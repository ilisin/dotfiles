# Dotfiles

My dotfiles.

---

## Prepare Install

```
sudo apt update
sudo apt install -y git
```

via ssh:
```
git clone git@github.com:nyogjtrc/dotfiles.git .dotfiles
cd .dotfiles
```

via https:
```
git clone https://github.com/nyogjtrc/dotfiles.git .dotfiles
cd .dotfiles
```

```
./pre_install.sh
```

## Install

```
./install
```
